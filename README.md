# 票据收款认领与异常挂账服务

任务类型：0-1代码生成

业务领域：纯后端服务

修改范围：跨模块多文件

任务难度：中等偏难

题面：

实现一个票据收款认领与异常挂账服务，财务导入银行回单后，业务人员按客户、金额和订单号认领收款，系统将回单与应收单核对并生成到账状态。正常流程要支持回单导入、自动匹配、人工认领、拆分认领、异常挂账处理和结清。同一回单重复认领、认领金额超过剩余金额、客户不匹配、已结清应收再次认领、异常挂账未处理直接结账都要拒绝。服务需要到账明细查询、挂账列表、应收单状态和对账导出，导出包含回单批次、认领人、差异原因和处理结果。重启后回单批次、认领状态、挂账处理和导出记录保持一致。验收要覆盖部分认领、撤销认领后余额恢复、自动匹配失败转挂账。
